VARIABELS, EXTEND & NESTING

SASS-דף דוגמה לשימוש בתכונות של

בעזרת שימוש נכון בתכונות כמו VARIABLES , EXTEND & NESTING אפשר להגיע לתוצאות דומות במהירות ויעילות רבה יותר.

1 - variables

.דוגמא לשימוש במשתנים בתרגיל: - עוזר במיוחד אם יש נתונים מבחוץ שנרצה לשלב

$globalSizeXXL:3.5em;
font-size: $globalSizeXXL;

$globalFontType: Verdana,
font-family: $globalFontType;

2 - Extend

.דוגמא לשימוש בהרחבה בתרגיל: - עוזר במיוחד ליצור מבנים של עיצוב שלמים ולקרוא להם אחר כך

%headBox {
width: 100%;
height: 200px;
text-align: center;
background-color: deepskyblue;

.header { @extend%headBox }

3 - Nesting

.דוגמא לשימוש בקינון בתרגיל: - עוזר במיוחד בכתיבה אחידה ומהירה של מבנה העמוד.

%headBox {
width: 100%;
height: 200px;
text-align: center;
background-color: deepskyblue;

h1 {
color: maroon;
font-size: $globalSizeXXL;
font-family: $globalFontType;
text-shadow: 2px 2px 2px green;

.CSS

.header {
width: 100%;
height: 200px;
text-align: center;
background-color: deepskyblue;
}

.header h1 {
color: navy;
font-size: 3.5em;
font-family: Verdana, Geneva, Tahoma, sans-serif;
text-shadow: 2px 2px 2px green;
}
